Install OpenClaw

One-command setup

Open Terminal and run the following commands to extract and install OpenClaw Auto.

Terminal 
# 1. Navigate to Downloads
cd ~/Downloads

# 2. Extract the package
unzip openclaw-auto-v1.0.zip

# 3. Enter the folder
cd openclaw-auto

# 4. Run installer (requires admin password)
./install.sh

🔐 Administrator Access

The installer will ask for your Mac password. This is normal — OpenClaw needs system access to run automation tasks. Your password is never stored or transmitted.

The installation process will:

  • ✓ Install OpenClaw core system
  • ✓ Set up AI model connection
  • ✓ Install all 20 workflows
  • ✓ Configure default settings

Troubleshooting

Common issues & fixes

Having trouble? Here are solutions to common issues.

❌ "Permission denied" when running install.sh

The installer needs execute permissions. Run: chmod +x install.sh then try again.

❌ "Command not found: openclaw" after installation

Restart your Terminal or run source ~/.zshrc (or ~/.bash_profile) to reload your PATH.

❌ Workflows not appearing in the menu

Try reinstalling the workflows: openclaw --reset-workflows

❌ AI features not responding

Check your internet connection and API key configuration. Run: openclaw --config to verify settings.

❌ Installation hangs or freezes

This may be due to slow internet during AI model download. Press Ctrl+C to cancel, then retry. The installer will resume from where it left off.
